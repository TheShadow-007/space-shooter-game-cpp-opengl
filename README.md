# 🚀 Space Shooter Game

A 2D Space Shooter Game developed in **C++** using **OpenGL (GLUT)**. Control a spaceship, destroy enemies and asteroids, survive boss battles, and achieve the highest score in an action-packed space environment.

## 🎮 Features

- Player movement and shooting controls
- Dynamic enemy and asteroid spawning
- Boss battles with health bar and attacks
- Collision detection and score tracking
- Shockwave power-up
- Explosion effects and animated space background
- Sound effects and background music
- Pause, restart, and game over functionality

## 🛠️ Tech Stack

- C++
- OpenGL (GLUT)
- Windows API (PlaySound)

## 🎯 Controls

- **W** – Boost Forward
- **A / D** – Move Left / Right
- **Space** – Shoot
- **M** – Shockwave Attack
- **P** – Pause
- **R** – Restart
